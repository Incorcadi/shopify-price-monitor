# Demo artifacts checklist

This folder is meant to hold **your** demo outputs for GitHub.

Recommended files:
- `current_prices.csv` — snapshot (generated by `price-monitor run`)
- `price_changes.csv` — diff (generated by `price-monitor run`)
- `slack_alert.png` — screenshot of Slack message (optional)

Suggested workflow:
1) Run baseline (creates DB + first snapshot)
2) Run again later (creates diffs if something changed)
3) Commit the demo artifacts you want to show

> Tip: If `price_changes.csv` stays empty for a long time, monitor a bigger scope, run a different collection,
> or use targets CSV mode to monitor products that change more often.
